# Pro SMS Backend Service

Hey, there! Welcome to the backend service of Pro SMS.

## Installation

1. First clone the repository.
   ```bash
   https://github.com/nexulyze/prosms-backend.git
   ```
2. Get the environment variables from any of collaborators
3. Run the following command to install packages
   ```bash
   yarn install
   ```
4. Now run the server in development mode
   ```bash
   yarn dev
   ```
